Methodology

  • Usage data: DeepSeek billing CSV export, June 1–30, 2026. 264,090 requests, 22,311,205,106 input tokens, 122,473,056 output tokens. Cache hit rate: 96.2% from billing data (input_cache_hit / total_input).
  • Pricing: OpenRouter model pages (June 30 – July 1, 2026). GPT/Claude pricing from official API docs (developers.openai.com, platform.claude.com). All prices USD per 1M tokens. Cached pricing from published provider rates.
  • Cache hit rates: DeepSeek from billing data. MiMo & MiniMax from OpenRouter weighted average tables. OpenAI/Anthropic estimated based on cache architecture (prefix auto vs explicit markers), context window limits, and known behavior with dynamic agent conversations.
  • Token efficiency: GPT-5.5 from tech-insider.org. Kimi K2.7 from Lushbinary. Opus 4.7 from Anthropic's own blog post + user reports. Opus 4.6 vs GPT-5.5 from MindStudio. Other models estimated from architecture family.
  • Formula: Cost = (cache_hit / 1M × cached_price) + (cache_miss / 1M × sticker_input) + (output / 1M × sticker_output × efficiency_multiplier)
